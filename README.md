squabble
========

Word game in java using GWT

Getting started
----------------
    git clone https://github.com/davidmoten/sowpods.git
    cd sowpods
    mvn clean install
    cd ..
    git clone https://github.com/davidmoten/squabble.git
    cd squabble
    mvn clean install
    cd squabble
    mvn jetty:run

Go to http://localhost:9292
